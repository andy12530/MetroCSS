MetroCSS
========
一个简单的Metro 风格前端框架

## 主要文件 ##
### metrocss.css ###

- 布局采用12列，格栅系统（流布局后续会加入）
- 基本的文字与段落
- 按钮
- 表单
- 表格
- 列表
- Tag标签
- Metro砖块
- 分页
- 提醒

### metrocss.js ###

- placeholder组件
- 标签页组件
- 弹出消息(Popover)

## 兼容性 ##
IE7+ / Chrome / Firefox / Opera 测试通过。
